GitSync

Git Sync provides Roblox programmers a simple and efficient way to store and retrieve scripts to and from a cloud-based database (GitHub).

See details
  • Sharing scripts (examples (snippets), templates, portfolios, team projects, coding challenges, etc.)
  • Transferring script/directories between places/games (ex. You want to make a campaign with a consistent gameplay system, so you would otherwise have to either copy and paste the scripts into every place (level) or create a Roblox package and redo everything every time you update the system)
  • Backing up scripts/directories outside of Roblox quickly and easily
  • Saving scripts so that you can view and work on them outside of Roblox Studio on almost any device
  • Quickly interacting with GitHub (save/update/import a file or directory in a few clicks, and get back to coding as soon as possible)

This is a plugin, which means it operates directly inside of Roblox Studio. This makes it a good choice for developers who don't need any functions beyond those offered by the plugin (pushing, pulling, branching, file-viewing) and developers who like having a very quick and easy way to interact with GitHub.
